Projetos Recentes

RAG Concurseiro - IA para Estudos

Um assistente inteligente de estudos desenvolvido para acelerar a preparação de candidatos em concursos públicos. O projeto implementa a arquitetura RAG (Retrieval-Augmented Generation) integrada a um banco de dados vetorial ativo para fornecer respostas contextualizadas sobre planos e ciclos de estudo, técnicas de revisão e editais. Desenvolvido em Next.js e Tailwind CSS, apresentando uma interface fluida, responsiva e otimizada.

Sistema de Gerenciamento de Usuários

Um sistema de login completo com cadastro, atualização de dados e dashboard. Desenvolvido com o frontend em Angular extremamente responsivo e o backend em Node.js com o banco de dados MongoDB. Nesse projeto foram implementadas as melhores práticas de segurança com o uso do bcrypt para encriptar senhas, cors para proteger a aplicação de requisições suspeitas, authguard para proteger as rotas e docker para containerização e deploy juntamente com AWS garantindo escalabilidade.

Cardápio Digital integrado ao WhatsApp

Calculadora Responsiva

App de Lista de Tarefas
